# LLM Prompt Sequence for Social Media Content Automation

This document outlines a sequence of prompts designed to automate the creation of a content calendar and social media posts for a small business.

## 1. Define the Business and Goals

The first step is to define the business, its target audience, and its social media goals.

**Prompt 1: Business Profile**

```
Create a detailed business profile for a small business. Include the business name, industry, target audience, brand voice, and social media goals.

**Business Name:** [Your Business Name]
**Industry:** [Your Industry]
**Target Audience:** [Describe your target audience in detail]
**Brand Voice:** [e.g., friendly, professional, humorous]
**Social Media Goals:** [e.g., increase brand awareness, drive website traffic, generate leads]
```

## 2. Generate Content Ideas

Next, generate a list of content ideas based on the business profile.

**Prompt 2: Content Idea Generation**

```
Based on the business profile provided, generate 20 content ideas for social media. The ideas should be relevant to the target audience and aligned with the social media goals. Categorize the ideas into themes (e.g., educational, behind-the-scenes, promotional).
```

## 3. Create a Content Calendar

Now, create a content calendar for the upcoming month.

**Prompt 3: Content Calendar Creation**

```
Create a one-month content calendar based on the content ideas generated. The calendar should be in a table format with the following columns: Week, Day, Time, Platform, Content Idea, and Call to Action.

**Month:** [Month]
**Platforms:** [e.g., Facebook, Instagram, Twitter, LinkedIn]
```

## 4. Generate Social Media Posts

Finally, generate the social media posts based on the content calendar.

**Prompt 4: Social Media Post Generation**

```
Generate a social media post for each entry in the content calendar. The posts should be written in the brand voice and include relevant hashtags.

**Content Idea:** [Content Idea from Calendar]
**Platform:** [Platform from Calendar]
**Call to Action:** [Call to Action from Calendar]
```
