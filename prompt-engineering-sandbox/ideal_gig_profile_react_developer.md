# Ideal Gig Profile: React Developer

- **Core Skills and Technologies:**
    - React, Next.js, TypeScript
    - State Management: Redux, Zustand
    - Styling: Tailwind CSS, CSS-in-JS (Styled Components, Emotion)
    - Testing: Jest, React Testing Library
    - Tooling: Webpack, Vite, ESLint, Prettier
- **Project Types:**
    - Building interactive and responsive user interfaces for web applications.
    - Developing and maintaining design systems and component libraries.
    - Integrating with RESTful and GraphQL APIs.
    - Performance optimization and code refactoring.
- **Gig Platforms:**
    - Upwork, Toptal, LinkedIn
- **Red Flags and Dealbreakers:**
    - Vague project descriptions with undefined scope.
    - Unrealistic deadlines and low budgets.
    - Clients with a history of negative feedback or poor communication.
    - Projects requiring outdated or obscure technologies.
- **Budget and Rate Requirements:**
    - Minimum hourly rate: $75/hour
    - Minimum project budget: $5,000
- **Client Profile:**
    - Tech startups and established companies with a clear product vision.
    - Agencies looking for experienced developers to augment their teams.
    - Clients who value quality and are willing to invest in good development practices.
