# Stock Market Analysis

**Parent Workspace**: [Financial Advisors](../)  
**Workspace URL**: [https://paxai.app/messages/financial-advisors](https://paxai.app/messages/financial-advisors)  
**Update Frequency**: Daily (Previous Day Analysis)

---

## Purpose

Comprehensive analysis of the stock exchange covering the previous day's market activity. This analysis provides insights into market movements, top performers, and significant stories impacting equity markets.

## Analysis Coverage

### Key Metrics Tracked
- **Top Gainers**: Stocks with the largest percentage gains
- **Top Losers**: Stocks with the largest percentage declines
- **Volume Leaders**: Stocks with highest trading volume
- **Price Movers**: Significant intraday price movements
- **Momentum Indicators**: Stocks showing strong momentum trends

### Market Stories
- Breaking news impacting specific stocks or sectors
- Analyst upgrades/downgrades and their market impact
- Merger & acquisition announcements
- Regulatory developments affecting traded companies
- Earnings surprises and guidance changes

### Sector Performance
- Sector-level performance breakdown (Technology, Healthcare, Financials, etc.)
- Relative strength across market segments
- Rotation trends and sector leadership
- Industry-specific developments

## Deliverables

Each daily analysis includes:
- Executive summary of market performance
- Top 10 gainers and losers (with context)
- Volume and momentum analysis
- Major news events and their market impact
- Sector performance heatmap/summary
- Notable technical levels and chart patterns

## File Naming Convention

Reports are named using the date format: `YYYY-MM-DD-analysis.md`

Example: `2025-11-10-analysis.md`

## Workflow

1. **Data Collection**: Gather previous day's market data
2. **Analysis**: Identify key movers, trends, and stories
3. **Report Generation**: Create markdown-formatted analysis
4. **Repository Update**: Commit report to this folder
5. **Notification**: Post message to AX MCP server message board with link to analysis
