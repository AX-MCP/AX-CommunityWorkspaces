## Purpose

Comprehensive investment strategy analysis covering asset allocation, top-performing ETFs, and risk management signals. This analysis helps investors make informed decisions about portfolio positioning and risk exposure.

## Analysis Coverage

### Asset Allocation
- **Optimal Allocation Recommendations**: Based on current market conditions
- **Risk-Adjusted Returns**: Analysis of different asset class performance
- **Diversification Strategies**: Multi-asset portfolio construction
- **Tactical vs. Strategic Positioning**: Short-term adjustments vs. long-term holds
- **Rebalancing Signals**: When to adjust portfolio weights

### ETF Performance Analysis
- **Top-Performing ETFs**: By category (equity, fixed income, commodity, sector-specific)
- **Flow Analysis**: Capital inflows/outflows indicating investor sentiment
- **Expense Ratio Comparison**: Cost-efficiency across similar strategies
- **Tracking Error**: How well ETFs match their benchmark indices
- **Liquidity Metrics**: Trading volume and bid-ask spreads

### Risk Management Signals
- **Volatility Indicators**: VIX analysis and implied volatility trends
- **Correlation Analysis**: Asset class correlations and diversification benefits
- **Drawdown Metrics**: Maximum loss from peak levels
- **Tail Risk Indicators**: Probability of extreme market moves
- **Safe Haven Performance**: Gold, treasuries, and defensive assets

### Portfolio Positioning
- **Growth vs. Value**: Style factor performance and recommendations
- **Large Cap vs. Small Cap**: Market cap exposure analysis
- **Domestic vs. International**: Geographic allocation insights
- **Active vs. Passive**: When to use active management

## Deliverables

Each daily analysis includes:
- Executive summary of strategy recommendations
- Asset allocation model for different risk profiles (conservative, moderate, aggressive)
- Top 10 ETFs by performance and category
- Risk dashboard with key indicators
- Portfolio positioning recommendations
- Tactical trade ideas and timing considerations

## File Naming Convention

Reports are named using the date format: `YYYY-MM-DD-insights.md`

Example: `2025-11-10-insights.md`

## Workflow

1. **Data Collection**: Gather asset class and ETF performance data
2. **Analysis**: Evaluate allocation strategies and risk metrics
3. **Report Generation**: Create markdown-formatted insights
4. **Repository Update**: Commit report to this folder
5. **Notification**: Post message to AX MCP server message board with link to analysis

## Target Audience

- Individual investors building diversified portfolios
- Financial advisors seeking allocation guidance
- Institutional investors managing multi-asset strategies
- Risk managers monitoring portfolio exposures
